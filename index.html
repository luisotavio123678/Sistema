Mural de Recados com Banco de Dados

Sobre o Projeto

Este sistema foi desenvolvido para permitir o cadastro, armazenamento e gerenciamento de recados utilizando um banco de dados. Seu objetivo é facilitar o controle das mensagens de maneira simples, organizada e segura.

Funcionalidades

  • Cadastrar recados
  • Visualizar mensagens
  • Salvar informações no banco de dados
  • Interface simples e intuitiva
  • Preparado para deploy na Vercel

Tecnologias

  • HTML5
  • CSS3
  • JavaScript
  • PHP
  • Banco de Dados SQL
  • Vercel
